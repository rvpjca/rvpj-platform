## For Administrators

### Getting Started

1. **Login**: Navigate to https://rvpj.in/login
2. Enter your email and password
3. You'll be redirected to the dashboard

### User Management

#### Creating New Users

**Creating Staff Members:**
1. Go to Dashboard → Users & Roles
2. Click "Create New User"
3. Fill in details:
   - Name
   - Email
   - Password (they can change it later)
   - Role (ADMIN, PARTNER, MANAGER, or STAFF)
4. Assign permissions (or use default role permissions)
5. Click "Create User"

**Creating Client Accounts:**
1. Go to Dashboard → Users & Roles
2. Click "Create New User"
3. Fill in details:
   - Name
   - Email (optional)
   - PAN Number (required for client login)
   - Birth Date (required for client login)
   - Role: CLIENT
4. Share PAN and birth date with client for login

#### Managing Permissions

1. Go to Users & Roles
2. Click on a user
3. View their current permissions
4. Toggle permissions on/off
5. Save changes

**Available Permissions:**
- `/dashboard` - Dashboard access
- `/dashboard/users` - User management
- `/dashboard/client-documents` - Document management
- `/dashboard/live-chat` - Chat support
- `/dashboard/whatsapp/bulk` - WhatsApp messaging
- `/dashboard/tasks` - Task management
- `/dashboard/attendance` - Attendance tracking
- `/dashboard/settings` - System settings

### Managing Client Documents

#### Viewing Documents

1. Go to Dashboard → Client Documents
2. You'll see all documents uploaded by clients
3. Use search to find specific clients
4. Filter by status (Pending, Approved, Rejected)

#### Reviewing Documents

1. Click on a document to view details
2. Click "Download" to review the file
3. Update status:
   - **Approved**: Document is accepted
   - **Rejected**: Document needs revision
   - **Pending**: Under review
4. Status updates are visible to clients immediately

### Live Chat Management

#### Handling Client Chats

1. Go to Dashboard → Live Chat
2. View all active chat rooms
3. Click on a chat to open conversation
4. Read chat history
5. Type your response and send

#### Assigning Chats

1. Open a chat room
2. Click "Assign to Staff"
3. Select staff member
4. They can now handle the chat

#### Scheduling Meetings

1. Open a chat with client
2. Click "Schedule Meeting"
3. Enter meeting details:
   - Title
   - Date and time
4. Client receives notification
5. Meeting appears in both calendars

#### Starting Video/Voice Calls

1. Open a chat room
2. Click "Start Video Call" or "Start Voice Call"
3. Share meeting link with client
4. Join the Jitsi meeting room

#### Closing Chats

1. When issue is resolved, click "Close Chat"
2. Chat moves to closed status
3. Client can still start a new chat if needed

### WhatsApp Bulk Messaging

#### Setting Up WhatsApp

**First Time Setup:**
1. Go to Dashboard → WhatsApp Bulk
2. Click "Login & Show QR Code"
3. Open WhatsApp on your phone
4. Go to Settings → Linked Devices → Link a Device
5. Scan the QR code on screen
6. Wait for "Connected" status

#### Sending Bulk Messages

1. Go to "Broadcast" tab
2. Enter phone numbers (one per line or comma-separated)
   - Format: +919999999999 or 9999999999
3. Compose your message
4. Preview recipients
5. Click "Send Broadcast"

#### Creating Message Templates

1. Go to "Templates" tab
2. Click "Create Template"
3. Enter:
   - Template name
   - Message content
4. Save template
5. Use template in broadcasts

#### Scheduling Messages

1. Go to "Scheduled" tab
2. Click "Schedule Message"
3. Enter recipients and message
4. Select date and time
5. Click "Schedule"
6. Message will be sent automatically

**Note:** WhatsApp session may disconnect. If disconnected, log in again with QR code.

### Task Management

#### Creating Tasks

1. Go to Dashboard → Tasks
2. Click "Create Task"
3. Fill in details:
   - Title
   - Description
   - Assign to (staff member)
   - Due date
   - Priority
4. Save task

#### Tracking Tasks

1. View all tasks in task list
2. Filter by:
   - Status (Todo, In Progress, Done)
   - Assigned to
   - Priority
3. Click task to view details
4. Update status as work progresses

### Attendance Management

#### Viewing Attendance

1. Go to Dashboard → Attendance
2. View attendance records
3. Filter by:
   - Date range
   - Staff member
4. Export reports if needed

#### Marking Attendance

Staff members can mark their own attendance:
- Clock In: Start of day
- Clock Out: End of day
- Break In/Out: During breaks

### Managing Enquiries

1. Go to Dashboard → Enquiries
2. View all contact form submissions
3. Click on enquiry to view details
4. Mark as resolved when handled
5. Follow up via email or phone

### Career Applications

1. Go to Dashboard → Career Applications
2. View all job applications
3. Click on application to view:
   - Candidate details
   - Resume (download)
   - Cover letter
4. Update application status:
   - Under Review
   - Shortlisted
   - Rejected
   - Hired

### System Settings

#### Site Settings

1. Go to Dashboard → Site Settings
2. Update:
   - Site name
   - Contact information
   - Social media links
   - SEO settings
3. Save changes

#### Theme Settings

1. Go to Dashboard → Theme Settings
2. Customize:
   - Primary color
   - Logo
   - Fonts
3. Preview changes
4. Save and publish

## For Clients

### First Time Login

1. **Receive Credentials**: Admin will provide your PAN number and birth date
2. **Visit**: https://rvpj.in/client-login
3. **Enter**:
   - Your PAN number (uppercase, e.g., ABCDE1234F)
   - Your birth date
4. **Login**: You'll be taken to your portal

### Client Portal Dashboard

After login, you'll see:
- Document status summary
- Recent activities
- Quick action buttons
- Upcoming meetings

### Uploading Documents

#### How to Upload

1. Click "Upload Document" or go to Upload page
2. Click "Choose File" or drag & drop
3. Enter document title (e.g., "Income Tax Return 2023-24")
4. Add description (optional)
5. Click "Upload"

#### Supported File Types

- PDF documents (.pdf)
- Images (.jpg, .jpeg, .png)
- Word documents (.doc, .docx)
- Excel sheets (.xls, .xlsx)

#### File Size Limit

- Maximum: 10 MB per file
- For larger files, contact staff via chat

### Viewing Your Documents

1. Go to "My Documents"
2. See all uploaded documents
3. Check status:
   - **Pending**: Under review by staff
   - **Approved**: Document accepted
   - **Rejected**: Needs resubmission
4. Download approved documents

### Using Live Chat

#### Starting a Chat

1. Go to "Live Chat" from menu
2. If no chat exists, one is created automatically
3. Type your message
4. Click Send

#### Chat Features

**Sending Messages:**
- Type text message and press Enter or click Send
- Be clear and specific with your queries

**Uploading Files:**
- Click attachment icon
- Select file to share
- File is sent to staff

**Video/Voice Calls:**
- Click video/voice call icon
- Join Jitsi meeting room
- Staff will join the call

**Viewing Meetings:**
- Scheduled meetings appear in chat
- Click "Join Meeting" at scheduled time

### Managing Your Profile

1. Go to Profile page
2. View your information:
   - Name
   - Email
   - PAN Number
   - Birth Date
3. Update contact details if needed
4. Change password (if available)

### Downloading Forms & Resources

1. Go to Downloads page
2. Browse available forms and documents
3. Click to download
4. Fill forms offline and upload back if needed

### Viewing Meetings

1. Go to "Meetings" page
2. See all scheduled meetings
3. Details include:
   - Meeting title
   - Date and time
   - Meeting link
4. Click "Join" when meeting time arrives

### Best Practices for Clients

#### Document Upload
- Use clear file names
- Upload only relevant documents
- Check file quality before uploading
- Don't upload duplicate files

#### Communication
- Be specific with your queries
- Respond promptly to staff questions
- Check chat regularly for updates
- Use professional language

#### Security
- Don't share your PAN/password
- Logout after each session
- Use secure internet connection
- Report suspicious activity

## FAQ

### General Questions

**Q: I forgot my password. How do I reset it?**  
A: Contact admin at admin@rvpj.in or call +91 9978781078 to reset your password.

**Q: Can I access the portal from mobile?**  
A: Yes! The platform is fully responsive and works on all devices.

**Q: Is my data secure?**  
A: Yes. All data is encrypted and stored securely. We follow industry-standard security practices.

### For Clients

**Q: What if my document is rejected?**  
A: Upload a corrected version or contact staff via chat to understand the issue.

**Q: How long does document review take?**  
A: Typically 1-2 business days. Check status regularly or contact via chat.

**Q: Can I delete uploaded documents?**  
A: No, but you can contact staff to request document removal.

**Q: The chat is not working. What should I do?**  
A: Refresh the page. If issue persists, contact us at +91 9978781078.

### For Staff

**Q: How do I request additional permissions?**  
A: Contact your manager or admin to request access to specific features.

**Q: Can I assign tasks to other staff?**  
A: Only managers and admins can assign tasks. Staff can only update their own task status.

**Q: How do I handle multiple chats?**  
A: Prioritize based on urgency. You can ask manager to reassign chats if overloaded.

### Technical Issues

**Q: Page is not loading.**  
A: Clear browser cache, refresh page, or try different browser.

**Q: File upload is failing.**  
A: Check file size (max 10MB) and format. Try uploading from different browser.

**Q: I can't see new messages in chat.**  
A: Refresh the page. If issue persists, try logging out and back in.

**Q: WhatsApp session keeps disconnecting.**  
A: This is normal. Simply scan QR code again to reconnect.
