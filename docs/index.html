Platform Documentation

Complete documentation for R V P J & Co. CMS Platform

What is this platform?

R V P J & Co. Platform is a comprehensive content management system designed for chartered accountancy firms.

Three Main Portals

1. Public Website

Marketing and information portal for visitors

2. Admin Dashboard

Staff and management portal for internal operations

3. Client Portal

Secure portal for client communication and documents

Platform Features

Public Website Features

  • Services information pages
  • 13+ Financial calculators (GST, Income Tax, EMI, SIP, RERA, etc.)
  • Knowledge bank and resources
  • Career portal with job listings
  • Contact and enquiry forms
  • Mobile responsive design

Admin Dashboard Features

  • User management with role-based access
  • Dashboard analytics and overview
  • Live chat management system
  • WhatsApp bulk messaging and broadcasts
  • Client document management
  • Content management system
  • Task and project management
  • Staff attendance tracking

Client Portal Features

  • Secure login with PAN & Birth Date
  • Document upload and management
  • Live chat with staff
  • Video/voice calls integration
  • Meeting scheduler
  • Download forms and resources

Getting Started

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Stable internet connection
  • For mobile: Android 8+ or iOS 12+

Login URLs

Public Website: https://rvpj.in

Admin Login: https://rvpj.in/login

Client Login: https://rvpj.in/client-login

First Time Setup

For Admin/Staff

  1. Receive your email and password from system administrator
  2. Go to https://rvpj.in/login
  3. Enter your credentials
  4. Click 'Sign In'

For Clients

  1. Receive your PAN number and birth date details
  2. Go to https://rvpj.in/client-login
  3. Enter PAN number (uppercase, e.g., ABCDE1234F)
  4. Enter birth date
  5. Click 'Login'
